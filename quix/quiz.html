6. What is the difference between a for loop and a while loop in C?

A for loop is used when the number of iterations is unknown, while a while loop is used when the number of iterations is known.
A for loop has three parts (initialization, condition, and increment), while a while loop only has a condition.
A while loop is executed at least once, while a for loop may not be executed at all if its condition is false from the start.
